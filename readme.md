# Blackjack Game in C

## 📌 Problem Statement
Implement a console-based Blackjack game using C.

## 🚀 Features
- Random card generation
- Dealer logic
- Win/Loss detection
- Input validation

## 🧠 Concepts Used
- Functions
- Loops
- Conditional statements
- Arrays
- Basic game loop design

## ▶️ How to Run
gcc src/main.c -o blackjack
./blackjack

## 📷 Sample Output
Player Cards: 10 | 7  
Dealer Cards: 9 | ?  
Hit or Stand?
